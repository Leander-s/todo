# Todo List Project

# Backend
Run
    
    ./backend_up

for a local test backend server running on localhost:8000.
Check [Backend-README](./backend/README.md) for further details.

# Frontend
Run

    ./frontend_up

for a local test frontend server running on localhost:5173.
Check [Frontend-README](./frontend/README.md) for further details.

# Database
For the database I use the default Django SQLite database. It runs with the backend server. For 
further details check [Backend-README](./backend/README.md).
